# 🍽️ SMILE CAFE POS - دليل التشغيل على اللابتوب والهواتف

## 📋 **نظرة عامة على النظام**

نظام SMILE CAFE POS هو نظام متكامل للمطاعم يتكون من:
- **خادم رئيسي** يعمل على اللابتوب/الكومبيوتر
- **تطبيق جرسونات** يعمل على الهواتف
- **شاشة مطبخ** تعرض الطلبات
- **شاشة كاشير** لإدارة الطلبات
- **قائمة رقمية** للعملاء مع QR Code

---

## 🖥️ **التشغيل على اللابتوب/الكومبيوتر**

### 1. **متطلبات النظام**
```bash
# Python 3.7 أو أحدث
# Windows/Mac/Linux

# المكتبات المطلوبة:
pip install flask flask-sqlalchemy qrcode[pil] pillow
```

### 2. **تشغيل النظام**
```bash
# الطريقة الأولى: تشغيل مباشر
python app.py

# الطريقة الثانية: استخدام دليل الإعداد
python setup_guide.py

# الطريقة الثالثة: استخدام الملف الدفعي
run_both_servers.bat
```

### 3. **الحصول على IP للشبكة**
```bash
# في ويندوز
ipconfig

# في ماك/لينكس
ifconfig
```

## 🔄 **سير العمل**

### **1. إعداد النظام**
1. تشغيل الخادم على اللابتوب
2. التأكد من اتصال جميع الأجهزة بنفس الشبكة
3. فتح التطبيقات المطلوبة على الأجهزة المختلفة

### **2. سير الطلب**
1. **الجرسون** يختار الطاولة ويأخذ الطلب
2. **الإرسال** الطلب يذهب مباشرة لـ:
   - شاشة المطبخ (للإعداد)
   - شاشة الكاشير (للمتابعة)
3. **المطبخ** يرى الطلب ويضغط "تم الإعداد"
4. **الكاشير** يرى حالة الطلب ويستلم الدفع
5. **الإشعارات** تظهر تلقائياً على جميع الشاشات

## 🛠️ **استكشاف الأخطاء**

### **مشكلة: الهاتف لا يصل للخادم**
```bash
# الحلول:
1. تأكد من اتصال الهاتف بنفس الشبكة
2. تحقق من جدار الحماية (Firewall)
3. استخدم IP صحيح
4. تأكد من تشغيل الخادم على 0.0.0.0:5000
```

### **مشكلة: الطلبات لا تظهر**
```bash
# الحلول:
1. تحقق من اتصال قاعدة البيانات
2. أعد تشغيل الخادم
3. تحقق من وحدة التحكم (Console) للأخطاء
```

### **مشكلة: الشاشة لا تتحديث**
```bash
# الحلول:
1. تحقق من اتصال الإنترنت
2. أعد تحميل الصفحة (Ctrl+F5)
3. تحقق من وحدة التحكم للأخطاء
```

## 📱 **نصائح للاستخدام الأمثل**

### **للجرسونات:**
- استخدم هاتف بشاشة كبيرة (5.5 بوصة فأكثر)
- تأكد من سرعة Wi-Fi جيدة
- احتفظ بشاحن الهاتف مشحوناً

### **للمطبخ:**
- استخدم جهاز لوحي (Tablet) إذا أمكن
- ضع الشاشة في مكان مرئي
- استخدم سماعات للإشعارات الصوتية

### **للكاشير:**
- استخدم كمبيوتر مكتبي أو لابتوب
- احتفظ بالصفحة مفتوحة طوال اليوم
- تحقق من الطلبات بانتظام
