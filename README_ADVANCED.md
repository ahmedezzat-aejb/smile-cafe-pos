# 🌟 SMILE CAFE POS - الإصدار المتقدم

نظام نقطة بيع متكامل لمقهى SMILE CAFE مع إدارة المخزون، تتبع الطلبات، وإدارة الموظفين.

## 🎯 **المميزات المتقدمة**

### 📱 **الوصول من جميع الأجهزة**
- **الهاردوير:** يعمل على اللابت توب والهاردوير
- **الموبايل:** متوافق مع جميع أجهزة الموبايل
- **الشبكة:** وصول من أي جهاز على نفس الشبكة
- **QR Code:** وصول سريع للعملاء عبر مسح الكود

### 🎨 **واجهات متعددة**
- **نقطة البيع:** واجهة سهلة للموظفين
- **لوحة التحكم:** لوحة تحكم متقدمة للمدير
- **قائمة العملاء:** قائمة رقمية للعملاء
- **QR Code:** كود QR للوصول السريع

### 💰 **إدارة الأسعار الذكية**
- **تحديث فوري:** تغيير الأسعار يظهر فوراً في القائمة
- **تحديث جماعي:** تحديث جميع الأسعار بنسبة مئوية
- **تتبع التغييرات:** سجل كامل بجميع التغييرات
- **نسخ احتياطي:** حفظ تلقائي للبيانات

### 📊 **لوحة تحكم متقدمة**
- **إدارة المنتجات:** إضافة، تعديل، حذف المنتجات
- **تحديث الأسعار:** تحديث فردي أو جماعي
- **QR Code:** إنشاء وتحميل QR Code للقائمة
- **التقارير:** تقارير يومية، أسبوعية، شهرية
- **الإعدادات:** تحديد نسبة الخدمة واسم المطعم

## 🚀 **التثبيت والتشغيل**

### الطريقة 1: التثبيت السريع
```bash
# تشغيل ملف التثبيت التلقائي
install.bat
```

### الطريقة 2: التثبيت اليدوي
```bash
# تثبيت المكتبات المطلوبة
pip install qrcode[pil] pillow flask flask-sqlalchemy

# تشغيل الخادم المتقدم
python server.py
```

## 🔗 **الوصول إلى النظام**

### من الجهاز المحلي
- **نقطة البيع:** http://localhost:8000/index.html
- **لوحة التحكم:** http://localhost:8000/admin.html
- **قائمة العملاء:** http://localhost:8000/menu.html
- **QR Code:** http://localhost:8000/qr.html

### من الأجهزة الأخرى
1. **العنوان IP:** استخدم IP الجهاز الذي يشغل الخادم
2. **QR Code:** امسح QR Code للوصول المباشر
3. **الشبكة المحلية:** استخدم `http://[IP]:8000`

## 📱 **الوصول من اللابت توب والهاردوير**

### اللابت توب
- يعمل على جميع أنظمة التشغيل (Windows, Mac, Linux)
- لا يتطلب تثبيت برامج إضافية على اللابت توب
- وصول كامل من متصفح الويب العادي

### الهاردوير/جهاز POS
- يمكن تشغيل النظام على أجهزة POS
- وصول عبر الشبكة المحلية
- متوافق مع شاشات اللمس

### الموبايل والتابلت
- واجهة متجاوبة بالكامل
- تعمل على جميع أحجام الشاشات
- تجربة مستخدم محسّنة للموبايل

## 📱 **QR Code للعملاء**

### المميزات
- **إنشاء تلقائي:** QR Code يتم إنشاؤه تلقائياً
- **تحديث فوري:** أي تغيير في الأسعار يظهر فوراً
- **طباعة:** يمكن طباعة QR Code للعملاء
- **تخصيص:** إعدادات QR Code متقدمة

### الاستخدام
1. العميل يمسح QR Code بكاميرا الهاتف
2. يتم توجيهه مباشرة إلى القائمة الرقمية
3. يمكن الطلب مباشرة من الهاتف
4. تحديث تلقائي كل 5 دقائق

- **حذف منتجات:** إزالة منتجات من القائمة

- **تحديث بنسبة:** زيادة أو نقصان بنسبة مئوية
- **تحديث جماعي:** تحديث جميع المنتجات دفعة واحدة
- **تخصيص:** نسب مخصصة للتحديث
- **إعادة تعيين:** العودة للأسعار الأصلية

### QR Code
- **إنشاء:** إنشاء QR Code جديد للقائمة
- **تحميل:** تحميل صورة QR Code
- **طباعة:** طباعة QR Code للعملاء
- **تحديث:** تحديث تلقائي للقائمة

### التقارير والإحصائيات
- **تقارير يومية:** إحصائيات المبيعات اليومية
- **تقارير أسبوعية:** ملخص الأسبوع
- **تقارير شهرية:** إحصائيات الشهر
- **تصدير:** تصدير التقارير كملفات

## 🔄 **التحديث التلقائي**
