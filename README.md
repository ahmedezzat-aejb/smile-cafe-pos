# نظام Smile Cafe POS
نظام نقطة بيع متكامل للمقاهي والمطاعم

![Smile Cafe Logo](https://via.placeholder.com/150x150/ff6b6b/ffffff?text=Smile+Cafe)

## 📋 الوصف
نظام Smile Cafe POS هو نظام إدارة متكامل للمقاهي والمطاعم، مصمم خصيصاً لتلبية احتياجات كافيه "Smile Cafe". يوفر النظام حلولاً شاملة لإدارة الطلبات، المخزون، الموظفين، والتقارير المالية.

## ✨ المميزات الرئيسية

### 🛒 نقطة البيع (POS)
- واجهة سهلة الاستخدام للطلبات
- دعم كامل للمنيو مع تصنيفات متعددة
- حساب تلقائي للخدمة والضرائب
- إدارة الطاولات والعملاء

### 📦 إدارة المخزون
- رط تلقائي للمكونات مع كل طلب
- تنبيهات للمواد المنخفضة
- إدارة الوصفات والمكونات
- تقارير المخزون التفصيلية

### 👥 إدارة الموظفين
- تسجيل الحضور والانصراف
- تتبع مبيعات كل موظف
- إدارة الورديات
- صلاحيات متعددة المستويات

### 📊 التقارير المالية
- تقارير المبيعات اليومية/أسبوعية/شهرية
- تحليل الأرباح والمصروفات
- تقارير المنتجات الأكثر مبيعاً
- تصدير التقارير بصيغة PDF/Excel

### 📱 الوصول عن بعد
- لوحة تحكم للمدير عبر الهاتف
- مراقبة حية للمبيعات
- تعديل الأسعار من أي مكان
- إشعارات فورية للأحداث المهمة

## 🚀 التثبيت والتشغيل

### المتطلبات الأساسية
- Python 3.8 أو أحدث
- نظام ويندوز/لينكس/ماك
- متصفح ويب حديث
- شبكة Wi-Fi للتشغيل على أجهزة متعددة

### خطوات التثبيت

1. **克隆 المشروع**
```bash
git clone https://github.com/yourusername/smile-cafe-pos.git
cd smile-cafe-pos
```

2. **إنشاء بيئة افتراضية**
```bash
python -m venv venv
# في ويندوز
venv\Scripts\activate
# في لينكس/ماك
source venv/bin/activate
```

3. **تثبيت المكتبات**
```bash
pip install -r requirements.txt
```

4. **تشغيل النظام**
```bash
python app.py
```

5. **الوصول إلى النظام**
- نقطة البيع: http://localhost:5000
- لوحة التحكم: http://localhost:5000/admin

### قاعدة البيانات
النظام يستخدم SQLAlchemy مع قاعدة بيانات SQLite للتشغيل المحلي:

- **Products**: المنتجات والأسعار
- **Inventory**: المخزون والمكونات
- **Orders**: الطلبات والفواتير
- **Employees**: بيانات الموظفين
